PREPT

An AI-Powered Mock Interview Marketplace

"Prept is a full-stack, AI-powered mock interview marketplace that bridges the gap between software engineering candidates and experienced engineers from top-tier companies like Google, Amazon, Meta, Microsoft, and Netflix. The platform runs a credit-based economy where interviewees book 1-on-1 sessions, and interviewers earn and withdraw credits — all powered by real-time HD video, persistent chat, and Google Gemini AI for live question generation and post-session feedback."

Problem Statement

Software engineering interview preparation is fragmented and ineffective. Platforms like LeetCode offer algorithmic practice but lack real human feedback and conversational dynamics. General-purpose video tools have no interview-specific features. Finding a FAANG engineer for a private mock interview is expensive and network-dependent. No single platform integrates scheduling, HD video, AI question generation, structured AI feedback, and a credit-based payment system in one product.

Market Research

The demand for quality technical interview prep has surged with rising competition for software engineering roles at FAANG companies. Platforms like Pramp, interviewing.io, and Exponent offer mock interviews but lack deep AI integration, real-time question generation, and a structured feedback engine. Prept captures this gap with a fully integrated, AI-first marketplace accessible to all engineers.

Research Gap / Innovation

Existing platforms either simulate interviews with AI bots (lacking human nuance) or connect peers without expert feedback. Prept's innovation lies in combining verified FAANG expert interviewers with Google Gemini AI for live question generation and post-interview structured feedback — delivering both human insight and AI precision within a single, credit-gated session economy.

System Methodology

1. Authentication & Onboarding

Clerk handles full authentication with JWT-secured sessions. On every login, checkUser() syncs the Clerk session to PostgreSQL, creating a new User record on first sign-in. The onboarding flow assigns roles (INTERVIEWEE / INTERVIEWER), collects profile bio and interview categories, and persists data via a Server Action. Clerk Billing detects the active subscription plan (free/starter/pro) on each session.

2. Credit-Based Economy

Three subscription tiers — Free ($0/1 cr), Starter ($29/5 cr), Pro ($69/15 cr) — gate platform features. Credits roll over monthly and are allocated atomically on login. All credit movements (booking deductions, interviewer earnings, withdrawals) are executed via Prisma database transactions to guarantee financial integrity with a complete CreditTransaction audit trail.

3. Slot Booking & Scheduling

Interviewers define a daily availability window; the system auto-generates 45-minute discrete slots, hides past slots, and marks booked slots as unavailable. The bookSlot() Server Action atomically validates credit balance, detects conflicts at database level, deducts credits from the interviewee, awards credits to the interviewer, and provisions Stream Video and Chat resources for the session.

4. HD Video Call Module

Each booking is linked to a unique Stream Video call ID. The Stream Video SDK powers 1080p HD video calls with screen sharing, automatic transcription (starts on first participant join), and session recording for Pro subscribers. Both parties are registered as 'host' members. The AI Question Panel is embedded directly within the call UI for real-time interviewer use.

5. AI Question Generator

Powered by Google Gemini 2.5 Flash Lite. The interviewer triggers the generator during a live session; a category-aware prompt (Frontend, Backend, DSA, System Design, Behavioral, DevOps, Mobile) returns 6 contextual questions with model answers in under 3 seconds. This eliminates question preparation overhead for interviewers and ensures session relevance for every category.

6. AI Feedback Report Engine

Post-session, the interviewer submits a structured 8-field form covering: Summary, Technical Depth, Communication Quality, Problem-Solving Approach, Strengths, Improvements, Recommendation (HIRE / NO_HIRE / CONSIDER), and Overall Rating. The data is stored in the Feedback table and presented to the interviewee as a detailed, structured AI feedback report accessible from the Appointments page.

7. Persistent Chat (Stream Chat SDK)

Every confirmed booking provisions a dedicated Stream Chat channel accessible before, during, and after the session. This enables pre-session preparation, real-time notes sharing during the call, and post-session follow-up — replacing the need for any external messaging tool and keeping all communication within the Prept platform context.

8. Security & Rate Limiting (Arcjet)

Arcjet's token-bucket algorithm enforces rate limiting on sensitive endpoints — fingerprinted by Clerk user ID rather than IP address to prevent bypass. The booking endpoint is capped at 5 requests/hour per user; the 6th attempt returns HTTP 429. All API routes are JWT-authenticated via Clerk. HTTPS is enforced across all routes with environment variables stored in Vercel.

Results & Analysis

AI Question Generation

Gemini 2.5 Flash Lite tested across all 8 categories. Average question generation time under 3 seconds per set of 6 questions with model answers. Category-aware prompts confirmed to produce highly relevant, role-specific questions.

Credit Economy Integrity

Prisma atomic transactions verified under concurrent booking simulation. Zero credit loss or double-deduction observed. Full CreditTransaction audit trail maintained across all booking, earning, and withdrawal operations.

User Acceptance Testing (UAT)

UAT conducted with 5 participants (3 interviewees, 2 interviewers) across 6 core tasks. All tasks accepted with an average satisfaction score of 4.6/5. Booking flow, feedback report, and AI question panel scored highest.

Security & Rate Limiting

Arcjet rate limiting confirmed: 6th booking attempt within 1 hour returns HTTP 429. Fingerprinting by Clerk user ID (not IP) prevents bypass via VPN or proxy. All 12 functional requirements implemented and validated.

Future Scope

Stripe Payment Integration

Direct Stripe integration for real-money credit purchases and interviewer payouts, replacing the current manual admin payout workflow with automated, instant transfers.

AI Interviewer Matching

A Gemini embedding-based recommendation engine that matches candidates to interviewers based on target company, role level, and historical feedback patterns for personalized interview preparation.

Collaborative Code Editor

In-call real-time collaborative code editor (CoderPad-style) for DSA and system design sessions, eliminating screen sharing for coding tasks and enabling execution within the platform.

Native Mobile App

React Native (Expo) cross-platform application providing a dedicated candidate experience with push notification reminders for upcoming sessions and on-the-go feedback access.

Interviewer Verification

LinkedIn OAuth-based employment verification to confirm that interviewers genuinely work at the companies listed on their profiles, building candidate trust and marketplace credibility.

Advanced Candidate Analytics

Dashboard tracking improvement across multiple sessions — rating trends, recurring weaknesses, target company readiness scores, and predicted hire probability based on feedback history.
